Для Windows большинство стратегий было взяты у [ Flowseal ](https://github.com/Flowseal/zapret-discord-youtube/), но сделал пару стратегий полностью под себя, со стратегиями знаете что делать.

Если стратегия на Linux для вас не сработала, рекомендую попробовать эту: https://github.com/un-couteau/zapret-conf-ufanet
