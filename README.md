Для Windows большинство стратегий было взяты у [ Flowseal](https://github.com/Flowseal/zapret-discord-youtube/), но сделал пару стратегий полностью под себя, рассширил домены для list-google.txt и добавил ещё листы, но только под свои стратегии. Со стратегиями знаете что делать. Ах да, стратегии не все взяты от Flowseal, так как удалил те, которые полностью неюзабельны для Уфанетовцев.

По поводу DNS. Я рекомендую вам использовать DNS от Яндекс, он быстрый и стабильный, а те сайты, которые необходимо обходить геолокацию добавлять вручную в hosts. Flowseal так же предлагал обходить Финляндию, для звонков в Дискорде, я не рекомендую добавлять в данный момент, так как в Уфанет всё работает без него и так, а после добавления наоборот можете сломать автоматический регион. 

Для Linux предлагаю только одну стратегию для оригинального zapret. Просто переместите все файлы в /opt/zapret, либо адаптируйте как вы посчитаете нужным.

Если стратегия на Linux для вас не сработала, рекомендую попробовать это: https://github.com/un-couteau/zapret-conf-ufanet

Так же из рекомендаций можете попробовать ещё и это: https://github.com/Sergeydigl3/zapret-discord-youtube-linux

Issues принимаю в том случае, если проблема с каким-то сайтом у вас возникла и проблема происходит и у меня. Прежде чем писать Issues - протестируйте все стратегии.
