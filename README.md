Для Windows большинство стратегий были взяты у [ Flowseal](https://github.com/Flowseal/zapret-discord-youtube/), но сделал пару стратегий полностью под себя, рассширил домены для list-google.txt, list-general.txt, ipset-all.txt и добавил ещё листы, но только под свои стратегии. Со стратегиями знаете что делать. Ах да, стратегии не все взяты от Flowseal, так как удалил те, которые полностью неюзабельны для Уфанетовцев.

По поводу DNS. Я рекомендую вам использовать DNS от [Яндекс](https://dns.yandex.ru/), он быстрый и стабильный(он к нам ближе всех в Уфе), а те сайты, которые необходимо обходить геолокацию - добавлять вручную в hosts. Flowseal так же [предлагал](https://github.com/Flowseal/zapret-discord-youtube/releases/tag/1.9.1) обходить Финляндию, для звонков в Дискорде, я не рекомендую добавлять в данный момент, так как в Уфанет всё работает без него и так, а после добавления наоборот можете сломать автоматический регион. Кастомный DNS обязательно нужно ставить в любом случае, для корректной работы zapret.

По поводу hosts. Это нужно добавлять для обхода разных геоблокировок у доменов, так же там есть домены для Дискорда, если всё таки у вас есть проблемы с работой автоматического региона, но необходимо их раскомментировать.

Путь hosts в Windows: C:\Windows\System32\drivers\etc\hosts

Путь hosts в Linux: /etc/hosts

Для Linux предлагаю только одну стратегию для оригинального zapret. Просто переместите все файлы в /opt/zapret, либо адаптируйте как вы посчитаете нужным.

Если стратегия на Linux для вас не сработала, рекомендую попробовать это: https://github.com/un-couteau/zapret-conf-ufanet

Так же из рекомендаций можете попробовать ещё и это: https://github.com/Sergeydigl3/zapret-discord-youtube-linux

Обход блокировок на Android (без Root):
1. Установить [ByeByeDPI](https://github.com/romanvht/ByeByeDPI/releases)
2. Зайти в настройки (шестерёнка) - использовать командную строку - Подбор стратегий (Beta)
3. В DNS поменять на 103.27.157.38 (для геоблокировок)
4. После тестирования можете применять стратегии, которые нашли

Issues принимаю в том случае, если проблема с каким-то сайтом у вас возникла и проблема происходит и у меня. Прежде чем писать Issues - протестируйте все стратегии.

P.S. Contabo провайдер не разблокирую вам, судя по всему это не РКН момент и не ТСПУ момент, а Уфанет решил заблокировать его. Обходить только любым VPN/Proxy. Так же в курсе о проблеме Steam при включении Zapret в Linux. Временное решение - сначала выключить Zapret, запустить Steam и потом включать его. 

По поводу Roblox, используйте альтернативный лаунчер [Bloxstrap](https://github.com/bloxstraplabs/bloxstrap/releases) или [Fishstrap](https://github.com/fishstrap/fishstrap/releases) для запуска плейсов через браузер. Это нужно для тех, у кого крашится Roblox после попытки входа в аккаунт.

По поводу Geometry Dash. Вам необходимо установить [Cloudflare Warp](https://developers.cloudflare.com/cloudflare-one/team-and-resources/devices/warp/download-warp/), чтобы устанавливать контент для уровней из newgrounds. Сначала запускаете Zapret, а потом запускаете Cloudflare Warp. Это так же может помочь с игрой Phasmophobia, если у вас с ней есть проблемы.
